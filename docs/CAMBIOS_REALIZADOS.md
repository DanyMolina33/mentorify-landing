# Cambios realizados

- Se creó un proyecto Vite Vanilla completo, sin React ni frameworks CSS.
- Se añadió hero cinematográfico con video, navegación, copy obligatorio y transición gradual a blanco.
- El formulario se convirtió en una tarjeta clara superpuesta, preservando IDs, validación, payload, webhook y flujo móvil de WhatsApp.
- Se añadieron bloques informativos de app y comunidad sin inventar destinos.
- Se mantuvo el orden original de autores, marquesina, FAQ, experiencia, oferta y popup.
- Se refinó el sistema visual con Inter, tokens centralizados, fondos blancos, bordes sutiles y responsive.
- La marquesina duplica sus elementos para un bucle continuo en dos direcciones.
- Se conservaron datos completos de autores/talleres, FAQ, precios, países, píxeles, Hotmart, cupón y voucher.
- Se añadió configuración centralizada y módulos ES para presentación, accesibilidad, reveal, comunidad, marquesina, oferta y tracking.
- Se documentó desarrollo, build y despliegue en Hostinger.
